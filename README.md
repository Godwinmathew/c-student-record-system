# Student Record Management System (C)

A simple console-based Student Record Management System built using **C programming language** and **file handling**.

## Features

- Add student record
- View all student records
- Search student by ID
- Delete student record
- Data stored using file system

## Technologies Used

- C Programming
- Structures
- File Handling
- Console-based interface

## How It Works

The program stores student data in a binary file:
students.dat


Users can manage student records using a menu-driven interface.

## Program Menu


Add Student

View Students

Search Student

Delete Student

Exit
