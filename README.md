# Student Record Management System (C)

A simple console-based Student Record Management System built using C programming language and file handling.

## Features
- Add student record
- View student records
- Search student by ID
- Delete student record
- File-based storage

## Technologies Used
- C Programming
- Structures
- File Handling

## How to Run
Compile and run the program, then select options from the menu.
